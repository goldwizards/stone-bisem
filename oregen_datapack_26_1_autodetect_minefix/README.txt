# 26.1 조약돌 생성기 업그레이드 광물 데이터팩 (자동 탐지 보강판)

## 지원 버전
- Minecraft Java Edition 26.1
- pack format: data pack 101.1

## 핵심 기능
- 플레이어 주변의 실제 조약돌 생성기 구조를 자동 탐지
- 조약돌 생성기 출력 칸을 marker로 자동 등록
- 주변 7x7 같은 높이 범위의 광물 블럭으로 레벨 업
- 블럭을 부수면 즉시 다운그레이드
- 새로 생성된 돌/조약돌에만 1회 확률 판정
- 6레벨에서 고대 잔해까지 등장

## 설치 방법
1. 이 zip 파일을 월드의 `datapacks` 폴더에 넣습니다.
2. 월드에 들어가서 `/reload` 를 실행합니다.
3. 채팅에 `[광물 생성기] 데이터팩 로드 완료` 가 뜨면 정상입니다.

## 사용 방법
- 생성기 근처에 가 있으면 약 10틱 주기로 자동 탐지합니다.
- 물과 용암이 붙어 있고 가운데가 돌/조약돌로 생기는 일반적인 생성기 구조를 감지합니다.
- 기존 수동 등록 명령도 남겨 두었습니다.
  - `/function oregen:player/register`
  - `/function oregen:player/unregister`
  - `/function oregen:player/info`

## 레벨 조건
- 1레벨: 조건 없음
- 2레벨: 석탄 블럭 7개
- 3레벨: 석탄 7 + 철 7
- 4레벨: 석탄 7 + 철 7 + 금 7
- 5레벨: 석탄 7 + 철 7 + 금 7 + 다이아 7
- 6레벨: 석탄 7 + 철 7 + 금 7 + 다이아 7 + 에메랄드 7

## 주의사항
- 생성기끼리 업그레이드 블럭을 공유하지 않게 하려고, 등록 시 다른 생성기와 너무 가까우면 막아 두었습니다.
- 대략 9블럭 이상 떨어뜨려 배치하는 것을 권장합니다.
- 자동 탐지는 플레이어 주변 범위에서만 수행됩니다.
- 업그레이드 블럭 판정은 매 10틱마다 갱신되고, 새 돌 생성 감지는 매 틱 처리됩니다.
- 광석은 `coal_ore`, `iron_ore` 같은 일반 광석으로 생성됩니다.
